Pathway Description


The information technology program is one of the three pathways at AET, with a focus on computer programming and broader technological skills. Students take one specialized computer science class each year, each one building on and expanding the skills learned in the previous year. These skills include Java programming, problem solving approaches, Python programming, interdisciplinary projects, the basics of machine learning, website design, and more.
